Initial Setup

  1. Scrum Master creates github repository from the starter code template
  2. Everyone clones the github repository
    git clone <SSH repo URL>
    (to clone a branch other than main
    git clone -b <branch name> <SSH repo URL>)
  3. Install the node module dependencies yarn install
  4. Create and checkout a working branch
    git checkout -b <working branch name>
    (could also be done with two commands
    git branch <working branch name>
    git checkout <working branch name>)
  5. Set up a github working branch
    git push -u origin <working branch name>
    (after that can update branch with git push)
  6. Run code yarn develop

Making a Pull Request

  1. Finish coding a feature on your working branch
  2. Stage changes on your working branch git add .
  3. Commit changes on your working branch git commit -m "message"
  4. Push commits to github working branch git push
    (if no github working branch
    git push -u origin <working branch name>)
  5. Make a pull request on github to merge github working branch into github main branch. There should NOT be conflicts. (If there are conflicts, complete steps 5 - 10 of "Updating Local branches" below and push the local working branch to the github working branch again)
  6. QA pulls the working branch
    git checkout --track origin/<working branch name>
  7. QA makes sure any new dependencies are installed yarn install
  8. QA runs code yarn develop
  9. QA tests feature and communicates with team about whether feature is ready or if changes are needed
  10. Scrum Master approves and completes the pull request on github

Update local branches after github main update

  1. Scrum master informs team that main branch has been updated
  2. Stages changes on your working branch git add .
  3. Commit changes on your working branch git commit -m "message"
  4. Push commits to github working branch git push
    (if no github working branch git push -u origin <working branch name>)
  5. Checkout your main branch git checkout main
  6. Update your main branch git pull origin main
  7. Test that code still works. yarn develop
    You may need to install new dependencies yarn install
  8. change branches git checkout <working branch name>
  9. Update your working branch git merge main
  10. Resolve any merge conflicts
